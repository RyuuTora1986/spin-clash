# Audio Drop Zone

Put generated files here with the exact filenames documented in:

- `docs/suno-audio-pack-20260423.md`

Folders:

- `music/` for looped BGM
- `sfx/` for one-shot UI and battle sounds
- `voice/` for short announcer lines

The current build now supports external music files from `music/`.
Use `mp3` filenames exactly as documented unless a later pass changes the runtime loader.
