# Spin Clash SUNO Audio Pack 2026-04-23

## 使用规则

- 下面每条都是单段 prompt，可直接整段复制到 SUNO。
- 每条已按 `500` 字符内思路压缩，不再拆正向 / 反向提示词。
- 风格总纲：霓虹、机械、竞技、短平快、高压碰撞，不要温柔、抒情、可爱、lo-fi、奇幻史诗。

## 场景节奏

- 主界面 / 选人 / 工坊：停留 `20-120` 秒，需要耐听循环。
- 战斗回合：常见有效聆听窗口 `10-30` 秒，前 `2-4` 秒必须进状态。
- 结算：更适合短 Sting、SFX、语音，不需要长 BGM。
