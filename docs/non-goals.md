# Non Goals

This project stage will not build:
- multiplayer
- PvP
- backend services
- login or accounts
- cloud save
- server-backed leaderboards
- in-app purchases
- app store packaging
- React or Vue migration without a proven need
- heavy runtime dependencies
- deep inventory or parts crafting
- large loot system
- open exploration
- free roaming map progression
- full roguelike buildcraft layer
- live-ops economy
- CMS or admin backend
- polished no-code editor
- invite reward economy
- social backend features
